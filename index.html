CYBER SCAPE

Welcome to CYBER SCAPE!


Cyber Space, provides you with an immersive experience of modern technology and world affairs. It is an experimental project by Aadit Garg and Tanmay Sharma of XI standard of S. D. Public School, Jagadhri under the guidance of Mrs. Geetanjli. It packs a powerful chatbot which shows the characteristics of a human with its vocal abilities, it generates a response based on the famous technique of checking and biases. On the other hand, there is a news-site, Newsroom-inc, which provides you with up-to-date news and information and updates every half an hour to keep one updated. There is also a third website which one can use to get Data about ANY book available online, for free.

Chat Bot

A chatbot is a computer program that simulates a human conversation. It can be used for various purposes, such as responding to a user's questions or providing information to a user. It mimics human behavior and communication style by artifitial means.

News App

A news app named Newsroom-inc is an app that provides you with the latest news and updates. It works by using API by GET POST Requests and refreshes itsellf evry half-an-hour or so. It provides with latest news from all over the world in every field.

Books

A website to extract data of any book available on the Internet for free, by using API and GET requests
